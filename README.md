## Описание проекта
Демонстрация применения принципов SOLID (SRP, OCP, LSP) в Java приложении.

## Реализованные принципы

### SRP (Single Responsibility Principle)
- `AreaCalculator` - только вычисление площадей
- `AreaPrinter` - только вывод результатов
- `ShapeFactory` - только создание объектов

### OCP (Open/Closed Principle)
- Интерфейс `Shape` позволяет добавлять новые фигуры без изменения существующего кода
- Легко добавить новые типы фигур (Triangle, Oval и т.д.)

### LSP (Liskov Substitution Principle)
- Все классы фигур взаимозаменяемы через интерфейс `Shape`
- Подклассы не нарушают поведение базового типа
